## Direction

- All concepts are full redesigns with larger DJ motifs (headphones, records, tonearm, deck bars).
- Colors stay toned down and brandable for stickers, socials, and print.

## Color directions (muted)

- Option 01: slate teal + cool gray
- Option 02: warm sand + charcoal
- Option 03: sage + soft stone
- Option 04: navy gray + dusty rose

## Figma import

1. Open Figma.
2. Drag these SVG files into your file/canvas.
3. Keep both variants per option on separate frames for review.
